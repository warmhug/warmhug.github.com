显示器各种接口 (<https://diy.pconline.com.cn/582/5823699_all.html>)

- VGA: 15孔、最广泛 最古老、模拟信号、1080P分辨率画面损失(可用在低于1080P分辨率显示器上)、只能传图像不能传声音。
- DVI: 综合好于VGA，1999年推出、接口巨大、模拟/数字信号、只能够传输图像，支持的设备很少。
- HDMI: 好于DVI，2002年推出、数字信号、支持4K分辨率、可同时传送音频和影像信号，主要有 Micro,mini HDMI和标准HDMI接口，但不能传输数据文件 (而普遍的安卓手机上的 Micro USB 接口可以传文件)。
- DP: 好于HDMI，2006年推出、数字信号、传输速率/支持分辨率都更大，支持3屏到6屏输出，苹果 Mini DP 接口更加强大。
- USB: 常见的 USB-A 接口类型，包含例如 USB 2 或 USB 3 的规格。(https://support.apple.com/zh-cn/HT201163)
  - USB 3 设备的功耗最高可达 900 mA（相比而言，USB 2 设备为 500 mA）。需要使用 USB 3 线缆（在插入电脑的接头上有“SS”符号），才能耐受 900 mA 的电流。USB 2 线缆不支持 900 mA。
  - USB 3.1 第 2 代最快能以 10 Gbps 的速率传输数据。
  - USB 3 和 USB 3.1 第 1 代最快能以 5 Gbps 的速率传输数据。
  - USB 2 最快能以 480 Mbps 的速率传输数据。
  - USB 1.1 最快能以 12 Mbps 的速率传输数据。

苹果电脑数据传输接口 (<https://support.apple.com/zh-cn/HT201736>)

- 雷雳 3 (USB-C): 2016 年起推出的 MacBook Pro 机型或更新机型。
- USB Type-C: 只是一种接口标准，接口更薄、可正反插、扩展性更强，但不代表任何速度或功能，而 雷雳3 与 USB-C 接口整合之后，最高可达 40Gb/s 的数据传输速率。而如果整合的是 USB 3.1 接口，数据传输速度是 5Gb/s 或 10Gb/s。Thunderbolt 3 连接线可以作为 USB 3.1 Type-C 连接线使用、而 USB 3.1 Type-C 连接线，却无法当做 Thunderbolt 3 连接线使用。苹果提供的 USB-C 连接线只能达到 USB 2.0 的传输速度，而达不到 Thunderbolt 3 的规范。Thunderbolt 3 连接线区分被动式的还是主动式的、主动式更好。
- 雷雳和雷雳 2: 2016 年之前的 MacBook Pro / Air 等机型。接口形状和 Mini DP 一样
